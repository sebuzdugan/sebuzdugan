## 🛡️ Responsible AI (FRAI)

- **[frai](https://github.com/sebuzdugan/frai):** Open-source toolkit for responsible AI. CLI + SDK to scan code, collect evidence, and generate model cards, risk files, evals, and RAG indexes.
- **[frai-benchmark](https://github.com/sebuzdugan/frai-benchmark):** A consensus "Panel of Experts" safety and compliance benchmark for SOTA LLMs (DeepSeek, Grok, GPT-5).
- **[frai-chat](https://github.com/sebuzdugan/frai-chat):** Responsible-AI copilot grounded in the EU AI Act, NIST AI RMF, and ISO/IEC 42001. BYOK, runs in the browser.
- **[frai-judge](https://github.com/sebuzdugan/frai-judge):** Model-as-judge evaluation used in the FRAI stack.
- **[framework-rai-mcp](https://github.com/sebuzdugan/framework-rai-mcp):** An MCP server that brings responsible-AI checks to agents.

More at **[frai.cc](https://www.frai.cc/)**.

## 🛠️ Practical tools

- **[AIEngineerCV](https://github.com/sebuzdugan/AIEngineerCV):** A CV builder for AI engineers. Web + CLI + Claude plugin, with an AI Recruiter Score.
- **[frontier-ai-prompts](https://github.com/sebuzdugan/frontier-ai-prompts):** A practical prompt library for GPT-5.5, Claude Opus, Gemini, and modern AI workflows.
- **[async-nlp-pipeline](https://github.com/sebuzdugan/async-nlp-pipeline):** A production-style FastAPI NLP service: async preprocessing, transformer sentiment, spaCy NER, rate limiting, SQLModel.
- **[chatbot-assessment](https://github.com/sebuzdugan/chatbot-assessment):** An AI chatbot for appointment scheduling.
